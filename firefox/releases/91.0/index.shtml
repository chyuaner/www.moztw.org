Firefox 新鮮事

  • 全新功能

    在全面 Cookie 保護的基礎上，增加了更全面的清除 Cookie 的邏輯，以防止洩漏隱藏的資料，並讓使用者易於了解哪些網站正在存儲本地資訊。我們強化了 Firefox 瀏覽器的隱私瀏覽模式，加入全面 Cookie 保護（Total Cookie Protection）功能，將 cookie 限制在產生他們的網站上，避免了解更多

  • 全新功能

    Firefox 現在支援使用 Windows 單一登入(SSO)登入 Microsoft、工作和學校帳戶。了解更多

  • 全新功能

    列印時的簡化頁面功能回來了！列印時，在 “更多設定” > “格式” 下選擇 “簡化” 選項（如果可用）以獲得整潔的頁面。了解更多

  • 全新功能

    HTTPS 優先策略：Firefox 隱私瀏覽視窗現在嘗試在所有網站使用安全連線，並且只在網站不支援時才使用非安全連線。了解更多

  • 全新功能

    新增了一個新在地化語言：低地蘇格蘭語。

  • 全新功能

    在隱私瀏覽視窗中，網址列搜索也提供「切換到該分頁」的功能。

  • 全新功能

    當在 MacOS 上選擇“增加對比”時，Firefox 現在會自動啟用高對比度模式 。

  • 全新功能

    Firefox 現在幾乎對所有使用者用互動執行catch-up paints，使大多數使用者用互動的反應時間縮短了 10-20%。
